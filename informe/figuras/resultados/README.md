# Figuras de resultados

- `sistema_anti.png`: detección histórica en modo `alerta`.
- `vm-muerta.png`: VM inoperable tras el simulador actual de procesos sin monitor.

Si dejas la captura en la raíz de `informe/` como `vm-muerta.png` (o `.jpg`),
muévela aquí con ese nombre:

```bash
mv informe/vm-muerta.png informe/figuras/resultados/vm-muerta.png
```

LaTeX referencia únicamente `figuras/resultados/vm-muerta.png`.
